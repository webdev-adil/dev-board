Question-1: What are the different ways to select an element in the DOM?

There are several ways to select an element in DOM. The choice of methods depend on the specific need and type of codes. Some of the most used methods are:

  1. getElementByID()
  2. getElementsByClassName()
  3. getElementsByTagName()
  4. querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent?

InnerHTML, innerText, and textContent are three properties used in Javascript to manipulate and extract text or HTML content inside an element. Nevertheless, these three properties work differently and serve different purposes.

innerHTML is used when the user needs to insert or extract any HTML content of an element. innerText is used when the user needs to extract the visible texts inside an element and ignore the hidden texts. Lastly, textContent is used to extract both visible and hidden texts inside an element.

Question-3: What is event delegation in the DOM?

Event delegation in DOM is a strategy in Javascript to attach an event listener to a parent element instead of attaching multiple event listeners to child elements. It works with a top to bottom approach where all the child elements get the property of event listener from the parent. Additionally, the elements added dynamically also get the property of the event listener from the parent element. As a result, the codes become more efficient and use less space and memory.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in Javascript, where an event starts at a target element. Later, it propagates upwards through the ancestor elements one by one and ends up in the root element or top most element in the chain.

Question-5: How do you create, add, and remove elements using JavaScript?

In Javascript, there are specific functions for creating, adding, and removing elements. These are discussed below:

1. Creating an element: document.createElement() is used to create an HTML element.
2. appendChild(), append(), prepend(), insertBefore(), etc. functions are used to insert elements into the DOM.
3. remove() and removeChild() functions are used to remove elements.
